# Extract Code Components

## Description

This tool extracts all instance of code components in a document and list them in the Output panel. The output panel can be saved as a file.

The following code types are supported:

- Any type
- ABNF (`abnf`)
- MIB (`mib`)
- YANG (`yang`, `yangtree`)
- XML (`xml`)

## Supported Document Types

- [x] RFCXML
- [x] Markdown
- [ ] Plain Text
