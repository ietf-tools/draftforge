# Expand/Fold/Peek References

## Description

This tool expand `<xi:include />` references into their full `<reference>...</reference>` entry. The reverse can also be done, folding `<reference>...</reference>` tags into `<xi:include />` references.

The **Peek Reference** function shows an inline preview of the full reference entry.

!!! tip
    This tool is triggered by clicking on **Expand Reference** or **Fold Reference into `<xi:include />`** link above references in the editor contents. The peek feature is triggered by clicking the **Peek Reference** link above references.

## Supported Document Types

- [x] RFCXML
- [ ] Markdown
- [ ] Plain Text
