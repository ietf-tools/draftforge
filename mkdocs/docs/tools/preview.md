# Open Preview

## Description

This tool opens a side-by-side preview of the current document. The preview is updated in real-time as content changes.

#### RFCXML

This tool depends on the `xml2rfc` system dependency.

Any warnings surfaced by `xml2rfc` will be listed in the Output panel.
